# Labdrug Pharmacy - Sales and Inventory Management System

A comprehensive web-based sales and inventory management system for Labdrug Pharmacy, built with Next.js and Neon DB.

## Features

- User authentication with role-based access (Admin, Manager, Cashier)
- Product management with categories and brands
- Sales transaction processing
- Inventory tracking with expiration date monitoring
- Low stock and expiring products alerts
- Sales reporting and auditing
- Customer management
- Supplier management
- Return processing

## Technology Stack

- **Frontend**: Next.js (React), Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: Neon DB (PostgreSQL)
- **Authentication**: JWT with bcrypt
